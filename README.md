# Ledgerly 💰

Ledgerly is a **Python Tkinter + MySQL finance management system**.

## 🚀 Features
- Dashboard with Income, Expense, Balance
- Income Management
- Expense Management
- Update & Delete Transactions
- PDF Invoice Export
- Financial Report PDF
- Search Transactions
- Recent Transactions
- Pie Chart Dashboard
- MySQL Database Support

## 🛠️ Technologies Used
- Python
- Tkinter
- MySQL
- Matplotlib
- ReportLab
- Jupyter Notebook

## ▶️ How to Run
```bash
pip install mysql-connector-python matplotlib reportlab
python ledgerly.py
```

## 📂 Project Files
- ledgerly.ipynb → Jupyter Notebook code
- ledgerly.py → Python executable file
- README.md → Project documentation
